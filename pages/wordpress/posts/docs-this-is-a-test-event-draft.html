How to create me

  • Create a new post
  • Do other page related things, like title, slug, etc.
  • Add the Event Pattern from the Gutenberg block library.
  • Fill in the Event details
  • Select the start and end date and time.
  • Yes… the date field is GMT and you have to enter it as GMT.
  • Fill in Location and Cost
  • There is also “Event Materials” if you need to add an agenda after the event.
  • Select “Event” as the template
  • Categorize as an Event




    Known issues
    GMT: We know it’s weird. This is something we can try to improve in the future, but also it’s how WordPress is designed and we are trying to be careful of adding too many non-core features this early on, so for now, please use GMT and when we sort out some other factors in the overall Design System, we can improve this.
